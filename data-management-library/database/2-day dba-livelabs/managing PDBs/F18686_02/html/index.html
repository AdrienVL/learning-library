Open PDBs Using EM Express

Before You Begin

Select the Oracle Database release:

This 10-minute tutorial shows you how to open PDBs using EM Express.

What Do You Need?

  • Oracle Database 18c19c
  • The container database (CDB) must be in read/write mode

Open a PDB

  1. In EM Express, ensure that you are in the Containers page for the ORCL CDB.
  2. In the Top Containers section of the Containers page, click the PDB that you want to open, and then choose Open/Close PDB > Open from the Actions menu. In this tutorial, you open PDB2.
  3. The Open PDB wizard appears and asks you to specify the open mode for the PDB (Read Write, Read Only, or Migrate), and whether you want the PDB opened in restricted mode. In this example, the PDB is opened in Read Only Restricted mode, so that access is allowed only to the PDB administrator. Click OK. Note: When you choose to open a PDB that is already open, the PDB closes immediately, and then reopens in the open and restricted mode that you specified in the wizard. When a PDB is closed immediately, users currently connected to the PDB are disconnected, and active transactions are implicitly rolled back.
  4. The PDB is opened. The Confirmation box appears and advises you that the PDB was opened successfully. Click OK.
    Note: In the EM Express Containers page, the up arrow icon appears in the Open Mode column for the PDB.

Open All the PDBs in a CDB

  1. In EM Express, ensure that you are in the Containers page for the ORCL CDB.
  2. In the Top Containers section of the Containers page, choose Open/Close PDB > Open All from the Actions menu.
  3. The Open All PDBs wizard appears and asks you to specify the open mode to use for all the PDBs (Read Write, Read Only, or Migrate), and whether you want the PDBs opened in restricted mode. In this example, the PDB is opened in Read Write Unrestricted mode, so that read write access is allowed to all the users. Click OK. Note: When you choose to open a PDB that is already open, the PDB closes immediately, and then reopens in the open and restricted mode that you specified in the wizard. When a PDB is closed immediately, users currently connected to the PDB are disconnected, and active transactions are implicitly rolled back.
  4. All the PDBs in the CDB are opened. The Confirmation box appears and advises you that all the PDBs were opened successfully. Click OK. Note: In the EM Express Containers page, the up arrow icon appears in the Open Mode column for all the PDBs.
